# Food-Insecurity-ML-model

Made at Columbia University


**What This Project Does:**
Instead of just looking at current food insecurity rates, this model calculates the trajectory (slope) of severe food insecurity over time for each country. It then uses macro-economic indicators (income inequality, poverty line headcount ratios, wealth distribution) to classify whether a country's situation is worsening or staying stable/improving.

Tech stack: Pandas, scikit learn, Kaggle, google colab
